# power-bi-data-professional-survey
Power BI Data Professional Survey Dashboard

Project Overview
This Power BI project analyzes survey responses from data professionals across different countries. 
The dashboard provides insights into salaries, job roles, career transitions, job satisfaction in the data industry.

Dataset Description
The dataset includes survey responses with the following information:
- Job Title
- Career Switch into Data
- Yearly Salary (USD)
- Industry
- Favorite Programming Language
- Job Satisfaction (Salary, Work/Life Balance, Management, Learning, etc.)
- Difficulty of Breaking into Data
- Age, Gender, Country, Education Level

Dashboard Insights
- Average salary by job role
- Salary distribution by country
- Most popular programming languages
- Career switch trends into data roles
- Job satisfaction levels across different factors
- Difficulty level of entering the data field

Tools Used
- Power BI Desktop
- Microsoft Excel
